Welcome to the LibreTexts and UC Davis JupyterHub!

This JupyterHub serves LibreTexts instructors and their students, as well as UC Davis faculty, staff, and students.

Need an account?

If you are a LibreTexts instructor or UC Davis affiliate, you can request an account by sending us an email from a Google Authentication connected email address. LibreTexts affiliated student accounts should be requested by their instructors.

Getting started with Jupyter

Jupyter is an environment where you can create interactive notebooks with code, visualizations, and more.

Some resources to learn Jupyter:

  • jupyter.org/try has interactive demo notebooks in many different languages
  • Familiarize Yourself with Jupyter, an example Jupyter Notebook tutorial on GitHub
  • Demo video of Jupyter of how to navigate the Jupyter Notebook.
  • Visit the JupyterLab documentation for more information on navigating the interface.

Any questions?

Please checkout our faq page. If you can't find your question, feel free to email us at jupyterteam@ucdavis.edu. For technical issues, open an issue on Github and we will try to resolve it.
